如何裁剪或生成eps格式图片

发布于 2020-02-19 | 分类于 LaTeX | 2分钟 | 416字数 | 浏览量：
  • EPS(Encapsulated PostScript)是PostScript的一种延伸类型。EPS（Encapsulated PostScript）是目前系统中功能最强的一种图档格式，向量及位图皆可包容。在使用LaTeX\LaTeXLATE​X编写文档插入图片时，eps或pdf都是个不错的选择。为了更好的掌控插入图片的大小，我们需要能够预览图片的大小。

1. Adobe Acrobat DC

在Adobe Acrobat中裁剪eps格式图片最为方便。

  • 只需要将在打开方式中选择使用Adobe Acrobat DC，便会自动为其创建pdf文件。
  • 编辑文件→\to→裁剪页面→\to→Enter→\to→导出到内嵌式PostScript(实际上可供保存的格式有很多)

2. GSview (Ctex套装自带，但也可以独立安装)

该软件古老小巧易用，在这里提供一个注册码 55555-24868

  • 用GSview打开eps格式图片
  • File →\to→ PS to EPS →\to→ Yes(不勾选Automatically calculate Bounding Box)
  • 分别选择图片左、下、右、下边界保存即可

3. Aode Illustrator

4. MATLAB保存eps格式图片

  • 一般的可以直接在图形窗口另存为eps格式的图片，这样保存的图片还有少许的空白区域，如果不需要则可以用上述方法进行裁剪。

  • saveas(gca, 'filename.eps', 'psc2')

    • 这种方法虽然可以保存eps格式图片并且保留原始颜色但含有大量多余的空白区域，还需要进行裁剪，不是很方便。

  • saveas(gca, 'filename.eps', 'eps')

    • 此种方法可以去掉空白区域，但是不能够保留颜色

  • print -depsc filename.eps

    • 此方法不仅可以去掉空白区域，还能够保留颜色，可谓最佳
